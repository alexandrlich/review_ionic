hooks are cordova folders for windows\unix systems:
unix systems = linux and mac


*how to use:
depends on yourPC  - overwrite your .cordova hooks with the ones provided in this folder. The main difference is double slash and support for ios platform


*limitatinos
hooks for windows don't support ios
hooks for linux don't support ios
hooks for mac supports android and ios


*examples of usage
cordova platform add android - d
cordova platform add ios -d
